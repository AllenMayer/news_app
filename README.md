# news_app

## Functionality:
* Rendering news list depending on user's locale
* Changing country
* Persistance using UserDefaults
* Showing full version in browser

## Stack:
* Swift
* MVVM
* URLSession
* JSON
* CocoaPods
* UserDefaults
* Autolayout
* Storyboard
* Programmatic UI
* WebView
* CountryPickerView https://github.com/kizitonwose/CountryPickerView
